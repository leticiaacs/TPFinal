Função que cria o cadastro.

Função que cria o cadastro de um novo usuário, utilizando dados inseridos de cpf e senha e retorna um inteiro para indicar se o cadastro foi bem sucedido ou não

Parameters
cpf_novo- string cpf com 11 caracteres inseridos pelo usuário para cadastro
senha_nova- string senha inserida pelo usuário para cadastro
Returns
inteiro - retorna 0 para um cadastro bem sucedido e 1 para erro no cadastro. Esses dados serão usados na main para instruir o usuario sobre os proximos passos, que dependerão do retorno desta função.

Função que verifica o login.

Função que compara o cpf e senha inseridos com o cpf e senha de usuarios cadastrados no banco de dados para identificar o usuário e emite uma mensagem de erro caso o cadastro não seja localizado. O throw da classe DadosInvalidos se encontra nesta função.

Parameters
c- string cpf de 11 numeros inseridos pelo usuário para fazer login
s- string senha inserida pelo usuário para fazer login
Returns
Usuario - retorna o usuário identificado pelo login, para ser usado em outras funções da main posteriormente
